# Antigravity Prompted App
This is a minimal React app created by Antigravity.
## How to run in StackBlitz
1.  Push this code to a GitHub repository.
2.  Prepend `stackblitz.com/github/` to your repository URL.
    *   Example: `stackblitz.com/github/username/repo-name`
3.  Hit Enter and StackBlitz will automatically install dependencies and start the dev server.
## Local Development
1.  `npm install`
2.  `npm run dev`
